 📖 Sobre o Projeto
Este repositório contém um script focado na manipulação de **Arrays (listas)** em JavaScript. O código demonstra, na prática, como adicionar, remover, inserir em posições específicas e ordenar dados dentro de uma estrutura de lista, fundamentos essenciais para qualquer aplicação.

 🧪 Conteúdo Técnico
push() Adição de elementos ao final do array.
unshift(): Adição de elementos ao início do array.
splice(): Método versátil para inserção, remoção e substituição de itens em índices específicos.
sort(): Ordenação alfabética dos elementos da lista.
console.table(): Exibição de dados em formato de tabela no terminal para melhor visualização.

🛠️ Como Executar
1. Certifique-se de ter o Node.js instalado.
2. Execute o comando no terminal:

node arrays.js

📖 About the Project
This repository contains a script focused on manipulating Arrays (lists) in JavaScript. The code practically demonstrates how to add, remove, insert at specific positions, and sort data within a list structure, which are essential fundamentals for any application.

🧪 Technical Content
push(): Adding elements to the end of the array.

unshift(): Adding elements to the beginning of the array.

splice(): Versatile method for inserting, removing, and replacing items at specific indices.

sort(): Alphabetical sorting of list elements.

console.table(): Displaying data in tabular format in the terminal for better visualization.
